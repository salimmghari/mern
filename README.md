### Launch:

#### Backend:

> - `cd backend/`
> - `npm install`
> - `npm run build`
> - `npm run start`

#### Frontend:

> - `cd frontend/`
> - `npm install`
> - `npm run web`
> - `npm run desktop`
> - `cordova platform add android`
> - `npm run android`
> - `cordova platform add ios`
> - `npm run ios`
